Day 2: Programming Intro

💻 Understood what programming is and explored the history and importance of C++.

🧠 What is Programming?
Programming is the art of instructing computers to perform tasks using logic and code. It helps solve real-life problems by automating work!

📜 History of C++:
C++ was created by Bjarne Stroustrup in 1983 as an extension of C. It introduced OOP concepts to make code more reusable and modular.

🚀 Importance of C++:
C++ is lightning fast ⚡ and close to hardware, which makes it ideal for game engines 🎮, operating systems 🖥️, and embedded systems 🔧.

🛠️ Compiler vs Interpreter:
- 🧾 Compiler: Converts the entire code into machine language before running it (e.g., C++)
- 🗣️ Interpreter: Translates code line-by-line at runtime (e.g., Python)

🌟 Tip: Start writing simple programs to practice logic, and always try to trace how each line works! 🧑‍💻✨
